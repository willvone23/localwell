# LocalWell

## Vision

LocalWell is a "healthy Yelp" — a platform for discovering restaurants, gyms, and wellness centers that support a healthy lifestyle. Deployed at localwell.app.

## Tech Stack

- **Frontend:** React 19 with Vite
- **Deployment:** Vercel (GitHub connected)
- **Domain:** localwell.app
- **Linting:** ESLint 9

## Commands

- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run lint` — run ESLint
- `npm run preview` — preview production build

## Key Features to Build

- **Business listings** — searchable directory of restaurants, gyms, and wellness centers
- **Health filters** — filter businesses by health-related criteria (dietary options, fitness types, wellness services)
- **Ratings & reviews** — user-driven ratings and review system
- **Social features** — user profiles, following, sharing, and community engagement

## Design Goals

Clean, modern wellness aesthetic. The UI should feel fresh, approachable, and health-oriented.
